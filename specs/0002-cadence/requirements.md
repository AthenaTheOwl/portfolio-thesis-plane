# Requirements — 0002 Cadence

The first non-calibration cycle. Spec 0001 anchored the rubric and
shipped the 2026-W25 calibration row. Spec 0002 makes the weekly
loop repeatable, lints the voice constraint, and adds the
decision-front-matter gate.

## Voice gate

- **R-PTP-011** `scripts/voice_lint.py` reads every `*.md` file under
  `reports/`, `decisions/`, and `docs/` and exits non-zero if any
  file contains a word from the `BANNED_FAIL` set defined in the
  script.
- **R-PTP-012** The `BANNED_FAIL` set covers marketing language and
  antithetical reversals as a structural device, per AGENTS.md.

## Decision gate

- **R-PTP-013** `scripts/validate_decisions.py` walks
  `decisions/DEC-PTP-*.md` and asserts every file has YAML
  front-matter with the keys `id`, `title`, `date`,
  `status: accepted|superseded`, and `supersedes` (nullable).
- **R-PTP-014** Decision filenames match the `DEC-PTP-NNN-<slug>.md`
  pattern; `id` in the front-matter matches the filename's
  `DEC-PTP-NNN` prefix.

## Weekly cadence

- **R-PTP-015** A second calibration row lands at
  `reports/2026-W26/`. The 2026-W26 cards are generated by the CLI
  (`python -m portfolio_thesis_plane generate --week 2026-W26`)
  from a `signals/2026-W26.yaml` curated against fresh signals.
- **R-PTP-016** The 2026-W26 rollup names the forced top-2 ATTEND
  and bottom-3 RETIRE. The list is allowed to change from 2026-W25
  but the mechanism is unchanged.

## Followthrough memo

- **R-PTP-017** `reports/2026-W26/_followthrough.md` names what
  happened to the 2026-W25 ATTEND picks. One paragraph per
  attended repo; "nothing" is a valid answer and is treated as
  evidence the rubric flagged the wrong repo.

## Gate set

By end of spec 0002 the gate set is:

```
python -m pytest
python scripts/voice_lint.py
python scripts/validate_schemas.py
python scripts/validate_registry.py
python scripts/validate_rubric.py
python scripts/validate_decisions.py
```

All six exit zero. This matches the v0-acceptance gate list named
in `specs/0001-foundation/acceptance.md`.
